Pagina listado

Aqui se podra ver la informacion de todos los clientes añadidos, ademas que se podra añadir un nuevo cliente, editar la informacion de un cliente ya existente y eliminar a un cliente ya existente

Añadir cliente

Puedes hacer clic en el boton: Añadir un nuevo cliente y este boton te llevara a la pagina para crear un nuevo cliente

Editar cliente

Al hacer clic en el boton: estaras entrando a una pagina en la cual podras ver la informacion de un cliente en especifico donde podras modificar los datos escritos ahi

Eliminar cliente

Existe el boton: que al pulsarlo se estaria eliminando el cliente y borrandolo de la base de datos

Nota importante

Al pulsar el boton eliminar, este no preguntara si se esta seguro de eliminar el cliente porque este al ser pulsado se eliminara automaticamente

Pagina Añadir cliente

En esta pagina se te pediran los datos de un nuevo cliente el cual todos los datos a rellenar seran obligatorios

Los datos que se pediran son:
  • Nombre

    • El nombre completo del cliente

  • Correo

    • correo electronico completo. Este correo debe contener todas las caractaresticas de un correo ex. correo@ejemplo.com

  • Numero de telefono

    • El numero de telefono debe ser compuesto por 13 caracteres. Con el codigo del pais, la extension del estado y el numero. Ex. +526144477836

  • Lugar donde vive

    • El lugar en el que recide el cliente actualmente

  • Nota

    • Una nota adicional en la cual se quiera especificar datos adicionales del cliente
